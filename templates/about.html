项目简介

本平台专为学生设计，提供词汇测试、错题归档、学习数据可视化等功能。 通过模块化架构与极简部署方式，快速落地教学场景。

核心创新点

  • 智能错题追踪：实时归档 + 正误分析
  • 动态可视化报表：周/月维度自定义标签
  • AI智能分析：读取用户信息准确帮助
  • 插件化设计：词表可插拔，前后端分离
  • 零配置部署：SQLite + 初始化脚本

技术架构

后端架构

  • 框架: Python Flask框架构建
  • 数据库: SQLite数据库存储数据
  • 架构模式: 完整的MVC结构

MVC分层设计

  • 模型层: 定义Student/QuizRecord等数据模型
  • 视图层: 多个路由处理函数
  • 控制层: 业务逻辑处理

主要功能模块

  • 用户系统: 登录/注册功能（含密码哈希加密）、会话管理（session机制）、邮箱验证功能（SMTP配置）
  • 学习模式: 自定义单词模式、四级/六级专项模式、随机题目生成功能
  • 数据管理: 错题本功能（记录错题及解析）、学习统计（UserStats模型）、单词管理（CustomWord/WrongWord模型）

技术亮点

  • 前端特性: 响应式布局设计、日间/夜间主题切换、加载动画效果、使用Tailwind CSS框架
  • 后端特性: 数据库迁移管理、路由拦截器（登录验证）、缓存控制头设置、配置文件分离
  • 插件化设计: 词表可插拔，前端可配置
  • 零配置部署: SQLite数据库 + 初始化脚本
  • 零配置部署: AI api 接入读取数据 智能分析

代码结构

  • 主程序(app.py): Flask应用初始化、数据库配置、路由定义、工具函数
  • 前端资源: HTML模板系统、CSS自定义样式、JavaScript交互逻辑
  • 数据库模型: 定义数据模型（Student/QuizRecord等）
  • 工具函数: 数据库操作、用户认证、数据处理等

扩展接口

  • AI服务集成: deepseek,豆包AI等
  • API配置: 未来展望火山方舟模型API、预留其他API接入点

该应用是一个功能完整的英语词汇学习系统，适合个人使用或教学场景，具有良好扩展性和现代化UI体验。
